# sprint10 - Análise Estratégica de Mercado de Restaurantes em Los Angeles

## Visão Geral
Neste projeto, conduzi uma análise exploratória de dados do mercado de restaurantes em Los Angeles com foco em:
- Estrutura do mercado (tipos de estabelecimentos)
- Presença de redes vs independentes
- Capacidade operacional (número de assentos)
- Distribuição geográfica (ruas com maior concentração)

O objetivo foi gerar insights estratégicos para tomada de decisão de investimento.

## Problema de Negócio
Um investidor deseja abrir um novo restaurante e precisa responder:

- Qual tipo de estabelecimento é mais viável?
- Vale mais investir em rede ou negócio independente?
- Qual localização maximiza o potencial de sucesso?
- Qual capacidade ideal (número de assentos)?

## Dataset
- 9.648 estabelecimentos analisados após limpeza
- Localização: Los Angeles
- Variáveis principais:
  - Tipo de estabelecimento
  - Pertencimento a rede
  - Número de assentos
  - Endereço

## Abordagem Analítica
- Limpeza e tratamento de dados
- Engenharia de features (extração de rua)
- Análise de distribuição de mercado
- Comparação entre redes vs independentes
- Análise de capacidade (assentos)
- Identificação de hotspots geográficos
